## 📋 개요 (Overview)

이 프로젝트는 Junyoung Yu의 전문 이력서를 GitHub Pages를 통해 공개합니다.
This project showcases the professional CV of Junyoung Yu through GitHub Pages.

**🌐 Live Site:** [https://swssmjy86.github.io/](https://swssmjy86.github.io/)

---

## 🛠️ 기술 스택 (Tech Stack)

- **Static Site Generator:** Jekyll
- **Markup:** Markdown, HTML5
- **Styling:** SCSS, CSS3
- **Hosting:** GitHub Pages
- **Language Composition:** SCSS (95.9%), HTML (4.1%)

---

## 📦 설치 및 실행 (Installation & Setup)

### 요구사항 (Requirements)
- Ruby 2.7.0 이상
- Jekyll 4.0 이상
- Bundler

### 로컬 실행 (Local Development)

```bash
# 저장소 클론 (Clone repository)
git clone https://github.com/swssmjy86/swssmjy86.github.io.git
cd swssmjy86.github.io

# 번들 설치 (Install dependencies)
bundle install

# 로컬 서버 실행 (Start local server)
bundle exec jekyll serve

# 브라우저에서 접속 (Open in browser)
# http://localhost:4000
